какие способы определения фунции сущ в JS и чем отличаются
1. Ключевое слово function
а) Можно вызывать до создания функции

2. Функциональное выражение
А) переменной будет присвоен результат функции
б) функцию можно вызвать только после того как она определена
В) можно не использовать название для функции

3. Стрелочные функции
а. Используются для быстрых выражений

4. Функция конструктор
А. Создается как обычная функция, но с заглавной буквы
Б. Небезопасная функция, такая данные передаются как строка

5. Метод объекта
А) могут быть объявлены как обычные функции так и стрелочные

структура http запроса ответа
Запрос:
1. Стартовая строка:
А. Метод запроса (Get, POST, Delete, put)
Б. URL -- адрес ресурса к которому идет обращение
В. Версия http протокола

2. Заголовки запроса -- словарь в котором передаются метаданные (версия браузера, IP, тип контент)
3. Тело запроса -- данные которые нужно отправить на сервер


Ответ:
1. Стартовая строка
А) версия протокола
Б) Код состояния (трехзначный код, который показывает результат обработки запроса)
В) Текстовое описание -- описание кода состояния
2. Заголовки ответа -- данные о сервере в виде словаря и возвещаемые данные
3. Тело запроса -- передается в зависимости от методов
Что передаем XML, JSON, форму данных
